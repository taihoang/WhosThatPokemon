# Who's That Pokémon?

A fun and interactive web game built with Next.js, React, and TypeScript. Test your Pokémon knowledge by identifying Pokémon from their silhouettes!

## Features

- Multiple difficulty levels (Easy: Gen 1, Medium: Gen 1-3, Hard: All)
- Multiple-choice questions
- Score tracking
- Smooth animations and transitions
- Pokémon sound effects
- Fully responsive design
- Accessibility features

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- **Next.js 14** (App Router)
- **React 18**
- **TypeScript**
- **PokeAPI** for Pokémon data

## Project Structure

```
├── app/              # Next.js App Router pages
├── components/       # Reusable React components
├── lib/              # Utility functions and API clients
├── types/            # TypeScript type definitions
└── styles/           # Global styles
```
